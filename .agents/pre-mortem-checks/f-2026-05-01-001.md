# Pre-Mortem Check: Dry-run inspection paths can still mutate tracked runtime aliases, making no-source-mutation validation untrustworthy.

Prevent this known failure mode:

- Pattern: Dry-run inspection paths can still mutate tracked runtime aliases, making no-source-mutation validation untrustworthy.
- Ask: When adding or reviewing a dry-run mode, did the validation prove it leaves all tracked files unchanged, including force-tracked runtime aliases?
- Do: After every dry-run proof command, run `git status --short` and fail if a tracked file changed; route dry-run artifacts to ignored temp or run archive paths.
