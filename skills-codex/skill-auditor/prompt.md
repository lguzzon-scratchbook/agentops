# skill-auditor

Two-pass audit of an existing SKILL.md against the unified AgentOps template. Pass 1 wraps heal-skill (7 structural-hygiene checks). Pass 2 adds 8 new content-discipline checks (description triggers, constraints front-loaded, rationale, verification checkpoints, output spec, quality rubric, references modularization, trigger clarity). Triggers: "audit skill", "skill quality review", "is this skill ready".

## Instructions

Load and follow the skill instructions from the sibling `SKILL.md` file for this skill.
Then read local files in `references/` and `scripts/` when needed.
